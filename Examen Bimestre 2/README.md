# Vertical Slice - Shooter 2D (Web / Phaser 3)

## Descripción
Vertical Slice jugable enfocado en mecánicas:
- Movimiento del jugador
- Disparo
- Enemigo con persecución simple
- Vida y feedback visual
- Condición de victoria: derrotar al enemigo

## Cómo ejecutar
Opción recomendada:
1. Abrir el proyecto con VS Code
2. Instalar/usar extensión **Live Server**
3. Click derecho en `index.html` → **Open with Live Server**

También puedes abrir `index.html` directo, pero Live Server evita problemas de rutas.

## Controles
- WASD o Flechas: mover
- Click izquierdo o Espacio: disparar

## Objetivo del Slice
Eliminar al enemigo y mostrar: "SLICE COMPLETADO".
