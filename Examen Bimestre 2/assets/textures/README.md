Añade aquí las imágenes de texturas que el juego usará.

Nombres de archivo esperados (colócalas en `assets/textures/`):

- buildable.png        -> textura para celdas construibles
- path.png             -> textura para celdas de camino
- occupied.png         -> textura para celdas ocupadas por torre
- tower_laser.png      -> textura/ícono para torre `laser` (id: laser)
- tower_plasma.png     -> textura/ícono para torre `plasma`
- tower_tesla.png      -> textura/ícono para torre `tesla`
- tower_missile.png    -> textura/ícono para torre `missile`
- tower_slow.png       -> textura/ícono para torre `slow`
- enemy_drone.png      -> textura/ícono para enemigo `drone`
- enemy_heavy.png      -> textura/ícono para enemigo `heavy`
- enemy_stealth.png    -> textura/ícono para enemigo `stealth`
- enemy_boss.png       -> textura/ícono para enemigo `boss`
- core.png             -> textura para el núcleo
- laser.png            -> textura/efecto de disparo láser (opcional)

Notas:
- El juego cargará automáticamente estas texturas si existen; si faltan, se usarán colores sólidos como fallback.
- Las imágenes pueden ser pequeñas (ej. 128x128) para iconos, y mayores para tiles (512x512) si quieres más detalle.
